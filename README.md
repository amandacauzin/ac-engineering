<em>Sobre o Projeto</em>

Este site, um misto de portfólio e catálogo de serviços, foi desenvolvido para facilitar a comunicação com o cliente, permitindo fácil acesso aos serviços ofertados.
Desenvolvido com foco em design responsivo, experiência do usuário e código limpo, este projeto marca o início da minha jornada como desenvolvedora Front-end.

**Objetivo:** Ofertar serviços de Engenharia Civil.
